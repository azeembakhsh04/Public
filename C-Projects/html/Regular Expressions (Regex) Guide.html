Regular Expressions (Regex) Guide

1. Literal Characters

What It Is: The exact characters you type.

Example: cat matches the word "cat" in the text.

2. Dot (.)

What It Does: Matches any single character except a newline.

Example: c.t matches "cat", "cut", "c2t".

3. Asterisk (*)

What It Does: Matches 0 or more of the preceding character.

Example: a* matches "", "a", "aaa".

4. Plus (+)

What It Does: Matches 1 or more of the preceding character.

Example: a+ matches "a", "aa", "aaa".

5. Question Mark (?)

What It Does: Matches 0 or 1 of the preceding character (makes it optional).

Example: colou?r matches "color" or "colour".

6. Square Brackets ([ ])

What It Does: Matches any one character inside the brackets.

Example: [abc] matches "a", "b", or "c".

7. Dash Inside Brackets (-)

What It Does: Specifies a range of characters inside brackets.

Example: [0-9] matches any digit from 0 to 9; [a-z] matches any lowercase letter.

8. Caret (^)

What It Does: Matches the start of a string.

Example: ^Hello matches "Hello" only if it's at the start of the string.

9. Dollar Sign ($)

What It Does: Matches the end of a string.

Example: world!$ matches "world!" only if it's at the end of the string.

10. Parentheses (( ))

What It Does: Groups characters or patterns together and remembers them.

Example: (abc)+ matches "abc", "abcabc" (one or more occurrences of "abc").

11. Pipe (|)

What It Does: Means "or" between patterns.

Example: a|b matches "a" or "b".

12. Curly Braces ({ })

What It Does: Specifies how many times the preceding character or group should appear.

Examples:
{n}: Exactly n times. Example: a{3} matches "aaa".
{n,}: At least n times. Example: a{2,} matches "aa", "aaa", "aaaa".
{n,m}: Between n and m times. Example: a{2,4} matches "aa", "aaa", "aaaa".

13. Backslash (\)

What It Does: Escapes special characters or indicates special sequences.

Examples:
\. matches a literal dot, not any character.
\d matches any digit (0-9).
\w matches any word character (letters, digits, and underscores).
\s matches any whitespace (spaces, tabs, etc.).

Quick Reference Summary

Regex Symbols and Uses:
.: Any single character except a newline.
*: 0 or more of the preceding element.
+: 1 or more of the preceding element.
?: 0 or 1 of the preceding element (optional).
[abc]: Any one of the characters a, b, or c.
[0-9]: Any digit from 0 to 9.
^: Start of a string.
$: End of a string.
(abc): Grouping; matches the exact sequence "abc".
a|b: Matches "a" or "b".
{2,4}: Matches between 2 and 4 of the preceding element.
\d: Matches any digit.
\w: Matches any word character.
\s: Matches any whitespace character.

Simple Use Case Example

Example: Suppose you want to find a phone number in a text that looks like this: "123-456-7890".

        
            let text = "Call me at 123-456-7890.";
            let regex = /\d{3}-\d{3}-\d{4}/;
            console.log(text.match(regex));  // Output: ["123-456-7890"]

Explanation:

  • \d{3}: Matches three digits.
  • -: Matches the literal dash.
  • \d{4}: Matches four digits.

This matches the pattern "123-456-7890" exactly.
